# TryHackMe Bank Lab Gobuster Exercise

This project shows how I used Gobuster to explore hidden folders and files on a simulated bank website in a TryHackMe lab.

The lab included files with unauthorized transactions that were not made by the bank. The goal was to practice web reconnaissance, find hidden folders, and respond to security issues.

## What I Did
- Ran Gobuster on the simulated bank website  
- Found hidden folders and files  
- Identified a file with unauthorized transactions  
- Returned the money to the rightful owner following lab instructions  
- Took screenshots to document the process  
- Learned how to find and respond to potential security problems

## Skills Practiced
- Web reconnaissance  
- Directory enumeration  
- Using command-line tools like Gobuster  
- Identifying unusual activity on websites  
- Writing clear documentation  
- Problem solving and incident response
